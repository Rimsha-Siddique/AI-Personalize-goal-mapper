**AI Personalized Goal Mapper**
AI Personalized Goal Mapper is a mobile application built with React Native and Expo that helps users create personalized goal roadmaps. The app uses AI (Google Gemini API) to generate goal-related questions and customized step-by-step roadmaps based on user input.
**Features**
User authentication (Sign up / Login)
AI-generated goal questions and roadmaps
Personalized roadmap creation
Editable roadmap steps
Save goals and roadmaps
Reminders and progress tracking                                           
Clean and user-friendly UI with animations
**Technologies Used**
React Native
JavaScript
Expo
Google Gemini API
GitHub
